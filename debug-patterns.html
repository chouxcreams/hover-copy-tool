Hover Copy Tool Debug Page

テスト用リンク:

User ID: 123 Post ID: 456 Product: 789 Local User: 999

推奨テストパターン:

  • 名前: "User ID", 正規表現: "user/(\d+)"
  • 名前: "Post ID", 正規表現: "post/(\d+)"
  • 名前: "Product ID", 正規表現: "product/(\d+)"

デバッグ手順:

  1. 拡張機能のポップアップを開いて、上記のパターンを1つ追加してアクティブにする
  2. ブラウザのデベロッパーツール（F12）でコンソールを開く
  3. 上記のリンクにマウスをホバーする
  4. コンソールログを確認する
